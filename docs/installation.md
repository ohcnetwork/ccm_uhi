# Installation

## Stable release

To install Care UHI, run this command in your terminal:

```sh
pip install Care UHI
```

## From source

The source files for Care UHI can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_uhi).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_uhi
```

Or download the [tarball](https://github.com/ohcnetwork/care_uhi/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_uhi/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_uhi
pip install .
```
